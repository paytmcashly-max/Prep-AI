# Launch Assets Checklist

Use this checklist to prepare store-ready screenshots, creative assets, and launch metadata for PrepAI.

## App Icon

- [ ] Final app icon exported at required platform sizes
- [ ] Android adaptive icon foreground prepared
- [ ] Android adaptive icon background color confirmed
- [ ] Icon is readable at small sizes
- [ ] Icon does not include misleading claims or restricted marks

## Splash Screen

- [ ] Splash screen uses final PrepAI branding
- [ ] Splash background matches app theme
- [ ] Splash asset renders cleanly on common Android screen sizes
- [ ] Splash asset renders cleanly on common iOS screen sizes

## Play Store Screenshots

- [ ] Phone screenshots prepared
- [ ] Screenshots use production-safe data
- [ ] Screenshots avoid real user data, tokens, emails, or resume content
- [ ] Screenshots do not mention internal backend or AI provider details
- [ ] Screenshots do not claim guaranteed jobs, interviews, or salary increases

## App Store Screenshots

- [ ] iPhone screenshots prepared
- [ ] Required screenshot sizes checked
- [ ] Screenshots use production-safe data
- [ ] Screenshots avoid real user data, tokens, emails, or resume content
- [ ] Screenshots do not mention internal backend or AI provider details
- [ ] Screenshots do not claim guaranteed jobs, interviews, or salary increases

## Feature Graphic / Promo Banner

- [ ] Play Store feature graphic prepared
- [ ] Promo banner copy is short and public-safe

- [ ] No unrealistic hiring or salary claims
